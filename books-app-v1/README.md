# 仕様書

- ユーザー管理は行わない
- 検索ボタン押下時、入力したテキストを元に、本の「タイトル」「著者」を対象に検索を行う
- 検索結果でヒットした本は、一覧表示する
- すべての本はデフォルトで「借りている」状態とする
- 「返却ボタン」押下で一覧表示から削除する
- 画面リロードで初期状態に戻る
